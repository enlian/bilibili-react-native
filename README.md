# ビリビリ React Native プロジェクト

このプロジェクトは、**React Native** を使用して開発されたビリビリ（Bilibili）風のモバイルアプリケーションインターフェースです。このプロジェクトは、最新の人気 **React Native** フレームワークやコンポーネントを活用して、リアルなユーザー体験を提供することを目指しています。**このプロジェクトは学習および交流の目的のみであり、商業目的での使用はできません。**

## 機能

- **React Native**（バージョン 0.75.3）を使用して開発
- ビリビリ風のユーザーインターフェースを実現
- 最新の **React Native** フレームワークやコンポーネントを使用
- 機能の改善と追加のために定期的に更新
- クロスプラットフォーム対応（iOS と Android）

## ページおよび機能概要

### 1. **ホーム（HomeTab）**

**機能**：
- 推奨動画の表示：おすすめの動画を一覧表示し、スクロールで参照可能。
- 動画カテゴリーナビゲーション：動画の種類ごとに分類し、簡単に動画を見つけられるようにします。
- 動画のプレビューと再生：動画のサムネイルをタップすると、詳細ページに移動し、再生できます。

**特徴**：
- 無限スクロールとページングをサポートし、快適なユーザー体験を提供。

### 2. **動画詳細ページ（VideoDetailScreen）**

**機能**：
- 動画再生：フルスクリーン再生、再生・一時停止、早送りなどの操作が可能。
- 動画の詳細：動画のタイトル、アップロード者情報、再生回数、いいね数などを表示。
- 弾幕やコメント：ユーザーは弾幕を送信したり、他の視聴者のコメントを確認したりできます。
- いいね、保存、シェア：動画に対するインタラクションをサポート。

**特徴**：
- 縦画面、横画面の切り替えが可能で、スムーズな操作が特徴。

### 3. **ダイナミクスページ（DynamicsScreen）**

**機能**：
- 動態の投稿：テキスト、画像、動画の投稿が可能。
- 動態リスト：ユーザー自身やフォロー中のユーザーが投稿した動態を表示し、いいねやコメントが可能。
- インタラクション通知：他のユーザーからのいいねやコメントを通知します。

### 4. **マイページ（My）**

**機能**：
- ユーザー情報の表示：フォロワー数、フォロー数、投稿動画数などのプロフィール情報を表示。
- 視聴履歴の確認：過去に視聴、いいねした動画を確認できます。
- アカウント設定：プロフィール情報の編集やアカウントの管理が可能。

### 5. **ボトムナビゲーションバー（TabNavigator）**

**機能**：
- ホーム、動態、個人情報ページへのクイックナビゲーションを提供。
- 動的な強調表示：現在アクティブなページのアイコンとテキストがハイライト表示されます。

### 特徴

- **クロスプラットフォーム対応**：iOS と Android の両プラットフォームで動作し、一貫したユーザー体験を提供します。
- **ユーザーインタラクション**：いいね、コメント、保存、弾幕などの操作が可能です。
- **定期的な更新**：React Native の最新技術に追随するため、プロジェクトは継続的に更新されます。

## 互換性

- **React Native**：バージョン 0.75.3

## インストールガイド

このプロジェクトを使用するには、以下の手順に従ってください：

1. リポジトリをクローン：

   ```bash
   git clone https://github.com/yourusername/bilibili-react-native.git
   cd bilibili-react-native
   ```

2. 依存関係をインストール：

   ```bash
   npm i
   ```

## アプリの実行

### iOS

iOS シミュレーターでアプリを実行するには、以下のコマンドを使用します：

```bash
npm run ios
```

このコマンドは iOS シミュレーターを起動し、アプリを実行します。

### Android

Android シミュレーターまたは実機でアプリを実行するには、以下のコマンドを使用します：

```bash
npm run android
```

このコマンドを実行する前に、Android シミュレーターまたは物理デバイスが接続されていることを確認してください。
